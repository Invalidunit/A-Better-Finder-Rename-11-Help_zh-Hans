    某些功能无法正常工作，我安装了 Unsity 应用程序增强器，安装了 FruitMenu，WindowShade X，Labels X 或其他任何功能。
 

安装了 Application Enhancer 后，所有措施均关闭。这个软件专门设计来破坏 macOS 的内置保护功能。

请尝试卸载它，随后查看问题是否仍然存在。

因为别人的错误而指责别人是很不礼貌的行为，因此在我们的辩护中，让我们引用 Apple 开发人员技术支持人员 George Warner 的话:

我们(苹果公司)的官方政策是不支持 APE。解析发送给我们的所有崩溃日志的数据收集器会自动忽略在回溯或 dylb 列表中具有 APE api 的任何报告。

同样，如果 DTS 在回溯或 dylb 列表中收到 API 导致的崩溃事件，我们将不进行调查。 在这种情况下，我们的"标准答案"是告知开发人员我们不支持 APE，并且只有在他们没有安装 APE 并且可以重现问题的情况下，我们才能为他们提供帮助。

我建议您对用户回复相同的内容. ;-)
