    如何将文件传输到运行 Windows 的电脑？
 

您可以使用 A Better Finder Rename 的"转换为有效的 NTFS/SMB 文件名"功能，以确保您的文件名对 Windows NTFS 卷有效。

在通过 SMB (远程安装的 Windows 驱动器) 连接将文件传输到 Windows 计算机时，您仍然可能会遇到一些问题。这些问题一般是由于两个因素造成的：隐藏的文件和 SMB 错误。

而你可以通过在传输文件之前使用 A Better Finder Attributes (A Better Finder Rename 的配套产品) 来删除隐藏的文件 ; 有时候 macOS 的 SMB 传输会受到 BUG 的干扰，虽然您一般不会遇到它们，如果确实遇到了这些问题，最好的做法是使用 Finder 中的"压缩"功能将要传输的文件压缩到一个 zip 归档文件中，然后向运行 WIndows 的电脑传输 zip 归档文件，并在 Windows 上解压 zip 文件。
