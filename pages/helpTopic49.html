    What are automatic software updates?
 

A Better Finder Rename contains an automatic software update feature.

By default this feature checks at regular intervals whether there is an update available on the website and if there is it will ask you whether you want to install the update or not. If you accept the update, it will be automatically downloaded from the website and installed.

This is a very easy way of staying up-to-date with the latest releases.

This feature is implemented using Andy Matuschak's Sparkle framework:

http://sparkle.andymatuschak.org/

Thanks Andy!
