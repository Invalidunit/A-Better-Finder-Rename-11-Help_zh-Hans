A Better Finder Rename 11 Help
    Saving File Lists
 

A Better Finder Rename allows you to save the current file selection in three different plain text formats:

  1. as a preview file (each line contains the current file name and new file name)
  2. as a simple template (each line contains the new file name)
  3. as an advanced template (each line contains the full POSIX path to the file and the new file name)
This feature is useful for tracking and documenting file name changes, or as input for the "Rename from file list" feature.

The preview and the advanced template formats both use the tab-separated text format that can be imported by many third-party database and spreadsheet programs.

You create a file list by selecting "Save File List..." in the "File" menu.

A dialog will appear allowing you to choose your preferred file format. You will then be asked for a name and a location for the new file.
